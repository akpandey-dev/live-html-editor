# Live HTML Editor – v1.0

Version 1.0 is the first stable implementation of the Live HTML Editor.  
It focuses on **direct browser-based learning**, live experimentation, and lightweight portability.

---

## Available Builds

### 1. Single-file Version (`single-file/`)

**index.html**

- Contains HTML, CSS, and JavaScript in one file
- Fully portable and works offline
- Ideal for immediate use, testing, or sharing

---

### 2. Modular Version (`modular/`)

**index.html + style.css + script.js**

- Code separated for better readability and maintainability
- Ideal for learning, extending, or contributing to the project

---

## Core Functionality

- Edit and run HTML, CSS, and JavaScript live in the browser
- Toggle JS and CSS editors individually
- Execute code in an iframe sandbox
- Export combined HTML, CSS, JS as a single downloadable file
- Optional file import for HTML, JS, and CSS
- Adjustable font size, word wrap, and night mode
- Easter egg / credits mode triggers a fun glitch animation
- Clear or reset editors individually or fully

---

## Browser Compatibility

Tested on Chromium-based browsers and Firefox.  
Requires JavaScript enabled.

---

## Design Notes

- No external libraries or frameworks
- Fully client-side; no backend required
- UI intentionally minimal to emphasize learning and experimentation more than a heavy feel


---

## Limitations

- Not for collaborative editing
- No advanced IDE features (debugger, autocomplete, version control)
